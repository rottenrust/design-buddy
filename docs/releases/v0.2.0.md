# Design Buddy v0.2.0

Первая тегированная публичная версия Design Buddy — персонального Telegram-наставника для начинающего UX/UI-дизайнера.

Вместо обычного свободного чата бот формирует профиль навыков пользователя и ведёт его через структурированные практические задания.

## Что появилось в v0.2.0

- Первичный последовательный опрос дизайнера.
- Постоянный профиль по 12 основным UX-навыкам.
- Каталог из 12 практических заданий в YAML.
- Пошаговый сценарий выполнения активного задания.
- Объяснение текущего шага, примеры и микро-задания.
- Проверка результата по заранее заданной рубрике.
- Сохранение подтверждений прогресса и обновление профиля навыков.
- Долговременная персональная память.
- Rolling summary и ограниченный контекст последних сообщений.
- Независимый запуск `codex exec` для каждого сообщения без постоянной agent-сессии.

## Техническая основа

- Python 3.12
- aiogram 3
- SQLAlchemy 2
- SQLite
- Pydantic
- Codex CLI
- pytest
- ruff

## Текущие ограничения

Версия рассчитана на персональное single-user использование. В неё пока не входят multi-user режим, веб-интерфейс, RAG, vector store, PostgreSQL и постоянные агентные сессии.
